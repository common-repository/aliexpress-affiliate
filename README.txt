Plugin for Aliexpress Affiliate

== Description ==

Aliexpress Affiliate helps you to add Aliexpress Affiliate link and image in your site.You have to configure plugin with required API details provided by Aliexpress. Shortcode is available to place ad in your page or post. You can also display Ad in sidebar using Text widget.

= Usage =
- Configure the plugin first. You will have to provide API Key, Tracking ID and Digital Signature.
- Use shortcode to display ad.
- Please update Default Category ID also. If no category or keyword is provided in the shortcode then Default Category ID will be used to fetch ads. See [this link](http://portals.aliexpress.com/help/help_center_API.html) to find out category ID.

= Shortcode Example =
- [ae_affiliate]
- [ae_affiliate width="250"]
- [ae_affiliate category="322"]
- [ae_affiliate category="34" keyword="motor cycle"]
- [ae_affiliate keyword="car non Slip mat"]

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'aliexpress-affiliate.zip' from your computer
1. Upload
1. Activate the plugin on the WordPress Plugin Dashboard
1. Go to Settings -> Aliexpress page and update required information.
1. Use shortcode as per your need.

= Using FTP =

1. Extract 'aliexpress-affiliate.zip' to your computer
1. Upload the 'aliexpress-affiliate' directory to your 'wp-content/plugins' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Go to Settings -> Aliexpress page and update required information.
1. Use shortcode as per your need.
